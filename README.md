Examples
```
node index.js upload --help

upload file:
node index.js upload -e https://s3.domain.com -a YVZO00SDGSSGDHK83D2 -s n4WFeOydWg4346o0XWTBJVL+xCDLRKacnOtTHd1v -b photo -p ./test/test.png

upload all files from dir:
node index.js upload -e https://s3.domain.com -a YVZO00SDGSSGDHK83D2 -s n4WFeOydWg4346o0XWTBJVL+xCDLRKacnOtTHd1v -b photo -p ./test

alternativ args usages:
npm start -- upload --help
npm run start -- upload --help
```

Build executable
```
npm run build
```
